ENABLE VIACAM 2.1.4.1 - WINDOWS X64
=====================================

Enable Viacam moves the mouse pointer using head movements captured by a
camera.

INSTALLATION
------------

Installer:
  Run eviacam-2.1.4.1-windows-x64-setup.exe.

Portable package:
  Extract the complete ZIP archive, keep all files together, then run
  eviacam.exe.

FIRST START
-----------

When several cameras are available, select the camera by its displayed name.
The default pointer settings are:

  Horizontal speed: 16
  Vertical speed:   15
  Acceleration:      3
  Smoothing:         2
  Movement threshold: 1

The pointer controller is enabled at startup. Keep a physical mouse or another
input method available during initial calibration.

WINDOWS SECURITY NOTICE
-----------------------

This community build is not digitally signed. Windows SmartScreen may display
a warning. Verify the SHA-256 checksum supplied next to the download before
running the application.

SYSTEM REQUIREMENTS
-------------------

  Windows 10 or Windows 11, 64-bit
  A Windows-compatible camera
